# BalatroOnline Backend

## 프로젝트 개요
- NestJS 기반의 온라인 보드게임(발라트로) 백엔드
- MySQL, Redis, WebSocket, Docker, Prisma ORM 등 현대적 스택 적용

## 폴더 구조
```
BalatroOnline/
├── src/                # 도메인별 코드 (user, auth, channel, room, lobby, game-history, redis, admin 등)
├── prisma/             # Prisma DB 모델
├── test/               # E2E 테스트
├── Dockerfile          # Docker 빌드 파일
├── docker-compose.yml  # 개발/운영 통합 환경
├── .env.example        # 환경변수 예시
├── package.json        # 의존성/명령어
└── ...
```

## 개발/운영 방법

### 1. 의존성 설치
```
pnpm install
```

### 2. 환경변수 설정
```
cp .env.example .env
# .env 파일에서 DB/Redis 정보 등 수정
```

### 3. DB 마이그레이션
```
npx prisma migrate dev --name init
```

### 4. 서버 실행
```
pnpm run start:dev
```

### 5. 테스트
```
pnpm test          # 단위 테스트
pnpm test:e2e      # E2E 테스트
```

### 6. Docker로 전체 실행
```
docker-compose up --build
```

## 주요 명령어
- `pnpm run start:dev` : 개발 서버 실행
- `pnpm run build`     : 빌드
- `pnpm test`          : 단위 테스트
- `pnpm test:e2e`      : E2E 테스트
- `npx prisma studio`  : DB GUI
